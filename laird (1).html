10-Day Swell Forecast
⚙ GFS-Wave Pipeline Required

The swell forecast timeline and hourly table require server-side processing of NOAA GFS-Wave GRIB2 files, which cannot be done in a browser. This is the same data source HopeWaves uses for its forecast.

What this section will show once the pipeline is running:
  1. A horizontal timeline chart showing swell height over 10 days as a line graph with area fill color-coded by swell window (■ green = in 115°–158° window, ■ orange = ±5° marginal, ■ gray = outside)
  2. Wind arrows every 3 hours above the timeline, color-coded by speed
  3. A "Worth a Check" row with ✓ marks where swell is green AND wind is light/offshore
  4. Low tide diamond markers with time labels (e.g., "low @ 6:45 AM")
  5. Nighttime shading between last light and first light
  6. An hourly forecast table below the chart with: Time, Height, Period, Direction (color-coded), Wind, and an expandable 2nd Swell column

To set up the pipeline:
  1. Create scripts/fetch_forecast.py — downloads GFS-Wave GRIB2 from NOMADS, extracts point data at 41.003°N, 71.600°W
  2. Create .github/workflows/update-forecast.yml — runs every 6 hours (cron: '30 3,9,15,21 * * *')
  3. Pipeline writes data/forecast.json with hourly height, period, direction, and secondary swell components
  4. This page reads data/forecast.json and renders the timeline + table

See the l__ai__rd Agent Spec for the complete pipeline code, NOMADS URL pattern, and xarray/cfgrib extraction script.

DeepZoom — Manual Setup Required

DeepZoom does not offer a public API for programmatic trip creation.
Each trip ID must be created manually through their website.

Setup (one-time, takes ~2 minutes):
1. Open deepzoom.com
2. Navigate to Fishers Island area (41.17°N, 71.82°W)
3. Enable current arrows (yellow layer)
4. Save as a trip — copy the trip ID from the URL
5. In index.html, replace the placeholder below:

src="https://www.deepzoom.com/trip/YOUR_TRIP_ID"
